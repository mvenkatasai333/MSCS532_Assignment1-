# MSCS532 Assignment 1 – Insertion Sort (Decreasing Order)
## Overview
This project implements the Insertion Sort algorithm in Python.  
The program sorts a list of numbers in **monotonically decreasing order** (from largest to smallest).
## How the Algorithm Works
Insertion Sort builds the sorted array one element at a time.
1. Start from the second element of the array.
2. Compare it with the previous elements.
3. Move smaller elements to the right.
4. Insert the current element in its correct position.
5. Repeat until the array is sorted.
## Example Input
[12, 5, 8, 19, 3, 7]
## Example Output
Original Array: [12, 5, 8, 19, 3, 7]  
Sorted Array (Decreasing Order): [19, 12, 8, 7, 5, 3]
